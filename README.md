# 🍳 煎蛋 Meme 风美化器

将 [煎蛋网](https://jandan.net/) 改造成 meme 社区风格。

## 主要改造

### 底部操作栏（参考右图）
每篇文章卡片底部添加操作栏，包含：
- ❤️ **点赞** - 悬停变红色
- 👎 **点踩** - 悬停变蓝色
- 💬 **评论** - 显示评论数
- ↗️ **分享** - 悬停变橙色

### 侧边栏（参考右图）
- 每个区块做成**独立圆角卡片**
- 带阴影和悬停效果
- 列表项带蓝色圆点前缀
- 标题带底部分割线

### 文章卡片
- 大圆角 (16px)
- 悬停浮起 + 阴影加深
- 缩略图圆角处理
- 分类标签彩色 pill

## 安装

1. 下载 ZIP 解压
2. Chrome 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择解压后的文件夹
